# UyaGUI v1 发布准备草案

## 当前定位

当前仓库已经具备：

- Phase 0 - Phase 5 的仓库内可编译/可测试/可 benchmark 基线
- Phase 6 的文档与 demo 应用骨架

但以下内容仍待真实硬件验证后再进入正式 `v1.0.0`：

- 板级兼容性验证
- 真实 DMA / GPU 后端验证
- 与 LVGL 的完整对比报告（文字渲染效果版已在仓库内补齐，完整性能/体积对比待补）

## 发布说明草案

- 新增完整组件库、动画、页面导航、资源系统、输入抽象
- 新增 Phase 5 性能测试与 benchmark 报告
- 新增 Phase 6 demo 应用、快速入门和移植文档

## 二进制分发建议

- 提供 `phase6_smoke` 宿主可执行产物
- 提供 `build-arm` 生成的 C99 交接产物
- 提供 `build-riscv` / `build-esp32` 的 microapp 产物

## 社区公告草案

一句话版本：

> UyaGUI 已完成从基础设施到示例应用的仓库内闭环，当前重点转向目标板验证与正式 v1 发布准备。
